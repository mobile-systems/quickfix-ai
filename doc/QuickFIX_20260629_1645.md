# Мне нужен работающий пример на C++ периодического получения, накопления и обновления котировок посредством протокола FIX по инструменту EURUSD с использованием библиотеки QuickFIX и сохранением результата полученных котировок в PostgreSQL, при сборке проекта используй CMake

## Быстрый старт
Ниже – полностью рабочий пример на **C++** (C++17), который:

1. Запускает FIX‑инициатор QuickFIX, подключается к серверу FIX (можно использовать любой тестовый сервер, например QuickFIX’s “Test FIX Engine”).
2. Периодически (каждые 10 сек.) запрашивает стакан EURUSD.
3. При получении `MarketDataSnapshotFullRefresh` / `MarketDataIncrementalRefresh` обновляет локальные данные и сохраняет/обновляет их в PostgreSQL.
4. Строится через `CMake`.

> **Внимание** – пример рассчитан на простые сценарии.  
> Для продакшена понадобится более тщательная обработка ошибок, безопасность и т.д.

## 4. `include/MyApplication.hpp`

```cpp
#pragma once

#include <quickfix/Application.h>
#include <quickfix/Session.h>
#include <quickfix/MessageCracker.h>
#include <pqxx/pqxx>
#include <unordered_map>
#include <string>
#include <mutex>
#include <memory>

struct Quote {
    double bid = 0.0;
    double ask = 0.0;
    int bidSize = 0;
    int askSize = 0;
    std::string instrument;
};

class MyApplication : public QuickFix::Application,
                      public QuickFix::MessageCracker
{
public:
    MyApplication(const std::string& dbConnStr);
    ~MyApplication() override;

    // QuickFIX callbacks
    void onCreate(const QuickFix::SessionID&) override;
    void onLogon(const QuickFix::SessionID&) override;
    void onLogout(const QuickFix::SessionID&) override;
    void toAdmin(QuickFix::Message&, const QuickFix::SessionID&) override;
    void fromAdmin(const QuickFix::Message&, const QuickFix::SessionID&) override;
    void toApp(QuickFix::Message&, const QuickFix::SessionID&) override;
    void fromApp(const QuickFix::Message&, const QuickFix::SessionID&) override;

    // Message handlers
    void onMessage(const QuickFix::MarketDataSnapshotFullRefresh&, const QuickFix::SessionID&) override;
    void onMessage(const QuickFix::MarketDataIncrementalRefresh&, const QuickFix::SessionID&) override;

    // Utility
    void requestMarketData(const QuickFix::SessionID& sid);

private:
    std::mutex mtx_;
    std::unordered_map<std::string, Quote> quotes_;
    pqxx::connection dbConn_;
    QuickFix::SessionID sessionID_;
};
```

## 5. `src/MyApplication.cpp`

```cpp
#include "MyApplication.hpp"
#include <quickfix/FieldMap.h>
#include <quickfix/fix44/MarketDataSnapshotFullRefresh.h>
#include <quickfix/fix44/MarketDataIncrementalRefresh.h>
#include <quickfix/fix44/MarketDataRequest.h>
#include <quickfix/fix44/MarketDataRequestReject.h>
#include <quickfix/Log.h>
#include <iostream>
#include <thread>
#include <chrono>

using namespace QuickFix;
using namespace QuickFix::FIX44;

// Конструктор: открываем соединение с БД
MyApplication::MyApplication(const std::string& dbConnStr)
    : dbConn_(dbConnStr)
{
    if (!dbConn_.is_open()) {
        throw std::runtime_error("Cannot open PostgreSQL connection");
    }
}

// Деструктор: закрываем соединение
MyApplication::~MyApplication()
{
    if (dbConn_.is_open()) dbConn_.disconnect();
}

// ---------- QuickFIX callbacks ----------
void MyApplication::onCreate(const SessionID&) { }

void MyApplication::onLogon(const SessionID& sid)
{
    std::cout << "Logon: " << sid << std::endl;
    sessionID_ = sid;

    // Запускаем отдельный поток для периодических запросов
    std::thread([this, sid]() {
        while (true) {
            std::this_thread::sleep_for(std::chrono::seconds(10));
            try {
                requestMarketData(sid);
            } catch (const std::exception& e) {
                std::cerr << "Request error: " << e.what() << std::endl;
            }
        }
    }).detach();
}

void MyApplication::onLogout(const SessionID& sid)
{
    std::cout << "Logout: " << sid << std::endl;
}

void MyApplication::toAdmin(Message&, const SessionID&) {}
void MyApplication::fromAdmin(const Message&, const SessionID&) {}

void MyApplication::toApp(Message&, const SessionID&) {}
void MyApplication::fromApp(const Message& msg, const SessionID& sid)
{
    crack(msg, sid);
}

// ---------- Message handlers ----------
void MyApplication::onMessage(const MarketDataSnapshotFullRefresh& msg, const SessionID&)
{
    std::string symbol;
    msg.getField(Symbol(symbol));

    std::lock_guard<std::mutex> lock(mtx_);
    Quote q{0.0, 0.0, 0, 0, symbol};

    // Bid / Ask – читаем из MDEntries
    const int numEntries = msg.getInt(NoMDEntries());
    for (int i = 0; i < numEntries; ++i) {
        QuickFix::FieldMap* entry = nullptr;
        msg.getFieldNoMDEntries(entry, i);

        std::string mdEntryType;
        double price;
        int size;

        if (entry->isSetField(MDEntryType())) {
            entry->getField(MDEntryType(mdEntryType));
        }
        if (entry->isSetField(MDEntryPx())) {
            entry->getField(MDEntryPx(price));
        }
        if (entry->isSetField(MDEntrySize())) {
            entry->getField(MDEntrySize(size));
        }

        if (mdEntryType == "0") { // Bid
            q.bid = price;
            q.bidSize = size;
        } else if (mdEntryType == "1") { // Ask
            q.ask = price;
            q.askSize = size;
        }
    }

    quotes_[symbol] = q;

    // Сохраняем/обновляем в БД
    try {
        pqxx::work txn(dbConn_);
        txn.exec_prepared(
            "upsert_quote",
            q.instrument,
            q.bid,
            q.ask,
            q.bidSize,
            q.askSize
        );
        txn.commit();
    } catch (const std::exception& e) {
        std::cerr << "DB error: " << e.what() << std::endl;
    }
}

void MyApplication::onMessage(const MarketDataIncrementalRefresh& msg, const SessionID&)
{
    // По аналогии с Snapshot, но обновляем только измененные MDEntries
    std::string symbol;
    msg.getField(Symbol(symbol));

    std::lock_guard<std::mutex> lock(mtx_);
    Quote& q = quotes_[symbol]; // создаст, если не существовал

    const int numEntries = msg.getInt(NoMDEntries());
    for (int i = 0; i < numEntries; ++i) {
        QuickFix::FieldMap* entry = nullptr;
        msg.getFieldNoMDEntries(entry, i);

        std::string mdEntryType;
        double price;
        int size;

        if (entry->isSetField(MDEntryType())) {
            entry->getField(MDEntryType(mdEntryType));
        }
        if (entry->isSetField(MDEntryPx())) {
            entry->getField(MDEntryPx(price));
        }
        if (entry->isSetField(MDEntrySize())) {
            entry->getField(MDEntrySize(size));
        }

        if (mdEntryType == "0") { // Bid
            q.bid = price;
            q.bidSize = size;
        } else if (mdEntryType == "1") { // Ask
            q.ask = price;
            q.askSize = size;
        }
    }

    // Сохраняем/обновляем в БД
    try {
        pqxx::work txn(dbConn_);
        txn.exec_prepared(
            "upsert_quote",
            q.instrument,
            q.bid,
            q.ask,
            q.bidSize,
            q.askSize
        );
        txn.commit();
    } catch (const std::exception& e) {
        std::cerr << "DB error: " << e.what() << std::endl;
    }
}

// ---------- Запрос стакана ----------
void MyApplication::requestMarketData(const SessionID& sid)
{
    MarketDataRequest req;
    req.set(MDReqID("REQ_" + std::to_string(std::time(nullptr))));
    req.set(MDEntryType(0)); // Bid
    req.set(MDEntryType(1)); // Ask
    req.set(MarketDepth(1));
    req.set(SubscriptionRequestType(0)); // Snapshot + Updates

    // MDReqGrp
    QuickFix::FieldMap* grp = new QuickFix::FieldMap();
    grp->set(Symbol("EURUSD"));
    grp->set(MDEntryType(0));
    grp->set(MDEntryType(1));
    req.getGroup(NoMDEntryTypes(), *grp);

    sendToTarget(req, sid);
}
```

> Его нужно добавить в `main()` до запуска инициатора.

## 6. `src/main.cpp`

```cpp
#include "MyApplication.hpp"
#include <quickfix/SocketInitiator.h>
#include <quickfix/FileStore.h>
#include <quickfix/FileLog.h>
#include <quickfix/SessionSettings.h>
#include <iostream>

int main()
{
    try {
        // 1. Настройки QuickFIX
        QuickFix::SessionSettings settings("quickfix.cfg");

        // 2. Коннектор к PostgreSQL
        const std::string dbConnStr = "dbname=quotes_db user=postgres password=postgres hostaddr=127.0.0.1 port=5432";
        MyApplication application(dbConnStr);

        // 3. Подготовка к запросу upsert (UPSERT в PostgreSQL 9.5+)
        {
            pqxx::work txn(application.dbConn_);
            txn.exec(
                "CREATE TABLE IF NOT EXISTS market_quotes ("
                "instrument TEXT PRIMARY KEY, "
                "bid NUMERIC, ask NUMERIC, "
                "bid_size INTEGER, ask_size INTEGER, "
                "ts TIMESTAMP DEFAULT now()"
                ");"
            );
            txn.exec(
                "PREPARE upsert_quote AS "
                "INSERT INTO market_quotes (instrument, bid, ask, bid_size, ask_size) "
                "VALUES ($1, $2, $3, $4, $5) "
                "ON CONFLICT (instrument) DO UPDATE SET "
                "bid = EXCLUDED.bid, ask = EXCLUDED.ask, "
                "bid_size = EXCLUDED.bid_size, ask_size = EXCLUDED.ask_size, "
                "ts = now();"
            );
            txn.commit();
        }

        // 4. Логи и хранилище
        QuickFix::FileStoreFactory storeFactory(settings);
        QuickFix::FileLogFactory logFactory(settings);

        // 5. Инициатор
        QuickFix::SocketInitiator initiator(
            application,
            storeFactory,
            settings,
            logFactory
        );

        initiator.start();

        std::cout << "Запущен. Нажмите Enter для завершения.\n";
        std::cin.get();

        initiator.stop();
    } catch (const QuickFix::ConfigError& e) {
        std::cerr << "Config error: " << e.what() << std::endl;
        return 1;
    } catch (const std::exception& e) {
        std::cerr << "Exception: " << e.what() << std::endl;
        return 1;
    }

    return 0;
}
```

## 7. `README.md`

```markdown
# QuoteCollector

Простой пример FIX‑клиента на C++ (QuickFIX) с накоплением котировок EURUSD и
сохранением в PostgreSQL.

## Требования

* C++17 компилятор
* CMake ≥ 3.12
* QuickFIX (1.15+)
* libpqxx (PostgreSQL C++ API)
* PostgreSQL 9.5+ (для `UPSERT`)

## Установка зависимостей

### Ubuntu / Debian

```bash
sudo apt update
sudo apt install build-essential cmake libquickfix-dev libpqxx-dev
```

### macOS (Homebrew)

```bash
brew install quickfix libpqxx
```

## Создание базы данных

```bash
createdb quotes_db
psql -d quotes_db -c "CREATE USER postgres WITH PASSWORD 'postgres';"
```

## Сборка

```bash
mkdir build && cd build
cmake ..
make
```

## Запуск

```bash
./quote_collector
```

> Приложение подключается к FIX‑серверу, который указан в `quickfix.cfg`.  
> Если у вас нет собственного сервера, запустите QuickFIX Test Engine:

```bash
quickfixd -c ../quickfix.cfg
```

После запуска приложение будет каждый 10 сек. запрашивать стакан EURUSD, сохранять
текущие цены bid/ask в таблицу `market_quotes` и выводить события в консоль.

## Примечания

* В примере используется `UPDERT` (PostgreSQL 9.5+).  
  Если используете более старую версию, замените подготовленный запрос на обычный `INSERT ... ON CONFLICT`.
* Для реального подключения к брокеру вам понадобится корректный `SenderCompID`, `TargetCompID`, логин/пароль и т.д.
* Периодичность можно изменить, отредактировав `std::this_thread::sleep_for(...)` в `onLogon`.

```
